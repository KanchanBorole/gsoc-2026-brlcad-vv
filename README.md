## Project Overview

Arbalest is a Qt based GUI application built on top 
of BRL-CAD that lets engineers run Verification and 
Validation checks on 3D geometry models.

This GSoC project focuses on two main goals:

1. Updating the existing V&V prototype to work with 
   the latest Arbalest and MOOSE codebase
2. Improving visualization, workflow, and reporting 
   aspects of the tool

---

## Project Goals

- Update existing V&V code to latest Arbalest codebase
- Separate V&V domain logic as a plugin/overlay
- Fix Qt5 to Qt6 compatibility issues
- Add TXT and JSON export formats
- Improve results panel with QTreeView grouping
- Add summary count bar
- Fix truncated Full Path display
- Improve viewport linking
